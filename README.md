## 📌 주요기능
+ 회원가입 페이지 : 아이디, 비밀번호, 이름, 성별 값 받기 및 유효성 검사
+ 로그인 페이지 : 로그인기능
+ pw찾기 페이지 : pw 변경 기능
+ 메인 페이지 : 현재 로그인한 사용자가 팔로워 하는 사용자 게시글 노출
+ 서치 페이지 : 사용자가 검색하고 싶은 다른 사용자 검색
+ 상세 페이지 : 사용자 프로필 사진 등 팔로우 및 게시글 관련 정보 노출
+ 게시물 만들기 페이지 : 게시글 작성 기능
+ 마이 페이지 : 현재 로그인한 사용자 정보 변경 가능
